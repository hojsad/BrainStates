      <h2>Fractional Occupancy</h2>
      <p><img data-src="Paper_FO.png"> LBD participants exhibited significantly higher FO in State 2 compared to both NC and AD suggesting it as a potential signature of LBD-specific neural dynamics.</p>

      <h2>Power Difference</h2>
      <img data-src="Paper_LBD_NC_power_diff_fdr.png" class="r-stretch"></section>
      <section id="lbd-subjects" class="slide level2">
      <h2>LBD Subjects</h2>
      <p>LBD subjects showed increased power in lower frequency bands (delta and theta) and reduced power in higher frequencies (alpha and beta) compared to AD, particularly in states 2 and 6.</p>
      </section>
      <section id="section" class="slide level2">
      <h2></h2>
      <ul>
      <li><em>Increases:</em> the left medial temporal cortex, left and right lateral temporal cortex, the left anterior cingulate and medial prefrontal cortex, right temporal-parieto-occipital junction, right inferior parietal cortex, and the left primary and early visual cortex.</li>
      <li><em>Decreases:</em> the right and left posterior cingulate cortex, cingulate motor areas, primary and early visual cortex, dorsal stream visual cortex, inferior somatosensory and motor cortex.</li>
      </ul>
      </section>

      <h2>Power spectral densities</h2>
      <img data-src="Paper_PSD.png" class="r-stretch"><p>A pronounced spectral slowing is evident in LBD, particularly in States 2 and 6.</p>

      <h2>TBR in LBD vs NC</h2>
      <img data-src="TBR_LBD_NC.png" class="r-stretch"><p>TBR elevations were observed in regions including the posterior and anterior cingulate cortex, medial prefrontal cortex, inferior parietal cortex, intraparietal sulcus, and posterior parietal gyrus.</p>
      </section>
      <section id="tbr-in-ad-vs-nc" class="slide level2">
      <h2>TBR in AD vs NC</h2>
      <img data-src="TBR_AD_NC.png" class="r-stretch"><p>TBR increases in AD compared to NC, particularly in the left primary and early visual cortices in States 2 and 3.</p>

      <section id="caf-scores" class="slide level2">
      <h2>CAF scores</h2>
      <p><strong>CAF scores</strong> showed consistent positive correlations with low-frequency activity (delta and theta) and negative correlations with higher-frequency bands (alpha and beta), particularly in States 2 and 6.</p>
      </section>
      <section id="caf-correlations" class="slide level2">
      <h2>CAF Correlations</h2>
      <ul>
      <li>In the <strong>occipital cortex</strong>, significant positive correlations were observed in the right primary and early visual cortex and the left ventral stream visual cortex. In contrast, the left primary and early visual cortex showed negative correlations in theta and alpha .</li>
      <li>In the <strong>anterior cingulate and medial prefrontal cortex (ACC/MPFC)</strong>, CAF positively correlated with theta power in both hemispheres and with delta power in the right hemisphere.</li>
      <li>In the <strong>lateral temporal cortex</strong> during State 6, alpha power negatively correlated with CAF.</li>
      </ul>

      function tippyHover(el, contentFn, onTriggerFn, onUntriggerFn) {
      const config = {
      allowHTML: true,
      maxWidth: 500,
      delay: 100,
      arrow: false,
      appendTo: function(el) {
      return el.closest('section.slide') || el.parentElement;
      },
      interactive: true,
      interactiveBorder: 10,
      theme: 'light-border',
      placement: 'bottom-start',
      };
      if (contentFn) {
      config.content = contentFn;
      }
      if (onTriggerFn) {
      config.onTrigger = onTriggerFn;
      }
      if (onUntriggerFn) {
      config.onUntrigger = onUntriggerFn;
      }
      config['offset'] = [0,0];
      config['maxWidth'] = 700;
      window.tippy(el, config);
      }
      const noterefs = window.document.querySelectorAll('a[role="doc-noteref"]');
      for (var i=0; i<noterefs.length; i++) {
      const ref = noterefs[i];
      tippyHover(ref, function() {
      // use id or data attribute instead here
      let href = ref.getAttribute('data-footnote-href') || ref.getAttribute('href');
      try { href = new URL(href).hash; } catch {}
      const id = href.replace(/^#\/?/, "");
      const note = window.document.getElementById(id);
      return note.innerHTML;
      });
      }
      const findCites = (el) => {
      const parentEl = el.parentElement;
      if (parentEl) {
      const cites = parentEl.dataset.cites;
      if (cites) {
      return {
      el,
      cites: cites.split(' ')
      };
      } else {
      return findCites(el.parentElement)
      }
      } else {
      return undefined;
      }
      };
      var bibliorefs = window.document.querySelectorAll('a[role="doc-biblioref"]');
      for (var i=0; i<bibliorefs.length; i++) {
      const ref = bibliorefs[i];
      const citeInfo = findCites(ref);
      if (citeInfo) {
      tippyHover(citeInfo.el, function() {
      var popup = window.document.createElement('div');
      citeInfo.cites.forEach(function(cite) {
      var citeDiv = window.document.createElement('div');
      citeDiv.classList.add('hanging-indent');
      citeDiv.classList.add('csl-entry');
      var biblioDiv = window.document.getElementById('ref-' + cite);
      if (biblioDiv) {
      citeDiv.innerHTML = biblioDiv.innerHTML;
      }
      popup.appendChild(citeDiv);
      });
      return popup.innerHTML;
      });
      }
      }
